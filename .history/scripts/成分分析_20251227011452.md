1️⃣ Head 与 Tail 的 POS 分布统计
| 角色             | 典型 POS                               | 举例                                                                                                                                             | 占比估计 |
| -------------- | ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
| **Head（前项）**   | VV（动词）                               | 睡/VV、洗/VV、吃/VV、看/VV、跑/VV、帮/VV、发/VV、请/VV、说/VV、聊/VV、唱/VV、跳/VV、打/VV、抽/VV、发/VV、道/VV、开/VV、离/VV、结/VV、考/VV、请/VV、放/VV、挂/VV、输/VV、减/VV、敬/VV、出/VV、回/VV 等等 | ≈95% |
| （极少）VA（形容词性动词） | 小/VA、大/VA、灰/VA 等（例如“小便”“大便”“灰心”）     | <5%                                                                                                                                            |      |
| 其他（Pseudo V）   | 少量 VV 派生，如提/VV（提醒），将/VV（将军），出/VV（出恭） | <2%                                                                                                                                            |      |

| 角色           | 典型 POS                                                     | 举例                                                                                                                                                                          | 占比估计 |
| ------------ | ---------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
| **Tail（后项）** | NN（名词）                                                     | 觉/NN、澡/NN、饭/NN、牙/NN、脸/NN、路/NN、步/NN、网/NN、书/NN、字/NN、门/NN、灯/NN、饭/NN、球/NN、烟/NN、面/NN、忙/NN、天/NN、话/NN、架/NN、婚/NN、病/NN、心/NN、假/NN、班/NN、学/NN、课/NN、事/NN、礼/NN、名/NN、鱼/NN、冰/NN、躬/NN、手/NN 等 | ≈96% |
| 其他（非典型名词）    | 少数 M（量词性词尾：针/M），或 VV（错误分词，如“理发”中发/VV），或 VA（“大便”“小便”中的便/VA） | <4%                                                                                                                                                                         |      |

2️⃣ 插入语（insertion_desc）主要类型与 POS 特征
| 类别              | 常见形式                                          | 对应 POS 标签     | 语法功能      | 举例                                       |
| --------------- | --------------------------------------------- | ------------- | --------- | ---------------------------------------- |
| **① 体标/时态助词**   | 了、着、过                                         | AS            | 体标        | 睡**了**一个好觉；跑**着**步；见**过**一面              |
| **② 数量短语**      | 一、两、三、几 + 次、个、天、年、瓶、支、斤、遍、趟、场、顿、声、手、页、口、通、根、份 | CD（数词）+ M（量词） | 数量表达      | 洗**了一把**脸；做**了一顿**饭；跑**了一会儿**步；发**了一通**火 |
| **③ 程度副词/时间副词** | 半天、一整天、刚刚、还、又、都、才、一直、终于                       | AD/NT         | 时间或程度     | 看**了半天**书；聊**了一整天**；刚刚订完婚                |
| **④ 修饰性形容词短语**  | 优美、丰盛、轰轰烈烈、重、老大、好、大、小、厚                       | JJ/VA/VV      | 修饰宾语或数量短语 | 跳**了一支优美的**舞；做**了一顿丰盛的**饭；受**了老大的**罪     |
| **⑤ 代词/名词性插入语** | 我、你、他、她、大家、自己、我们 + 的                          | PN / PN+DEG   | 受事、所有关系   | 帮**了我一个**忙；扫**了大家的**兴；领**我的**情           |
| **⑥ 指示词/限定词**   | 这、那、这个、那个                                     | DT            | 限定、特指     | 死**了这条**心；挑**了一担**水                      |
| **⑦ 副动词插入**     | 把、再生、互相、提、值、站好、灰、操碎、伤透                        | VV / AD       | 强调或复合结构   | 操**碎了**心；伤**透了**心；把**好**关；站**好**岗        |
| **⑧ 无插入**       | （无中间成分）                                       | —             | 紧连结构      | 吃饭、走路、散步（少见）                             |

3️⃣ 插入语常见模式总结（pattern_desc → pattern_type）
| pattern_type        | 典型形式           | 插入结构例子                                                    | 占比估计 |
| ------------------- | -------------- | --------------------------------------------------------- | ---- |
| **two_tokens**      | V + [插入词组] + N | 洗**了一把**脸 / 跑**了一会儿**步 / 做**了一顿**饭 / 见**过**一面 / 发**了一通**火 | ≈75% |
| **same_token（重叠式）** | AAB 式     | 散散步、聊聊天、睡睡觉、把一把脉                                           | ≈10% |
| **whole（整体词）**      | 连写成一个词         | 下班、吹牛、结婚、革命、出院                                            | ≈10% |
| **none / failed**   | 未识别出插入语或分词错误   | 理发、打针、造反、干杯 等                                             | ≈5%  |

4️⃣ 主要插入词 POS 分布统计（汇总）
| POS                    | 常见词汇                          | 功能    | 大致比例          |
| ---------------------- | ----------------------------- | ----- | ------------- |
| **AS（时态助词）**           | 了、着、过                         | 体标语气  | ~45%          |
| **CD/M（数词+量词）**        | 一、两、三、几 / 次、个、天、年、斤、场、顿、声、支、趟 | 数量结构  | ~40%          |
| **PN / DEG（代词所有格）**    | 我、你、他、大家、自己 / 的               | 所属或受事 | ~10%          |
| **JJ / VA / VV（形容词性）** | 优美、丰盛、轰轰烈烈、重、老大、好、大、小         | 修饰语   | ~7%           |
| **DT（指示词）**            | 这、那、这条、那次                     | 限定语   | ~3%           |
| **AD（副词）**             | 一下、半天、再、都、还                   | 状语修饰  | ~10% （常与体标共现） |
